std.d.lexer.StringCache
public struct StringCache
The string cache is used for string interning.
It will only story a single copy of any string that it is asked to hold. Interned strings can be compared for equality by comparing their .ptr field. Default and postbilt constructors are disabled. When a StringCache goes out of scope, the memory held by it is freed.
